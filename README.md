# cruise-data-quality-framework

## Overview
A Python-based data quality validation framework simulating a cruise reservation system with bookings, passengers, and payment transactions.

## Key Features
- Synthetic data generation (10,000 records)

## Tech Stack
Python, Pandas, SQL, Jupyter Notebook

## Data Model
- Bookings (Parent)
- Passengers (Child)
- Payments (Transactions)


## Disclaimer:
This project uses entirely synthetic data and is intended for educational and portfolio purposes only. 
It does not contain or reflect any proprietary or confidential information from past or current employers.
